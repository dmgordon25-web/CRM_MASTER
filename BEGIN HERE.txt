BEGIN HERE - CRM Tool
=====================

WHAT TO CLICK FIRST
-------------------
Double-click: RUN ME FIRST - Install CRM Tool.bat

WHAT HAPPENS NEXT
-----------------
The installer will:
1) Copy CRM Tool into %LOCALAPPDATA%\CRM Tool
2) Create a Desktop shortcut named exactly: CRM Tool
3) Launch CRM Tool when installation finishes
4) Print a success message in the installer window

HOW TO OPEN CRM AFTER INSTALL
-----------------------------
After installation, always use the Desktop shortcut "CRM Tool".
Do not start CRM from any old scripts or duplicate launchers.

WHAT THE OTHER FILES/FOLDERS ARE
--------------------------------
- crm-app: the CRM web app files used by the launcher
- scripts: internal maintenance/build/install scripts (not for normal daily use)
- docs: project documentation
- tests: automated test files
- devtools/tools: internal developer utilities
- Start CRM.bat: internal launcher used as the shortcut target after install

TROUBLESHOOTING
---------------
- If install fails, open: %TEMP%\CRM_Tool_Install.log
- If CRM does not open from the shortcut, run RUN ME FIRST - Install CRM Tool.bat again
- Ignore old or duplicate run/install scripts if you still have them from previous copies
